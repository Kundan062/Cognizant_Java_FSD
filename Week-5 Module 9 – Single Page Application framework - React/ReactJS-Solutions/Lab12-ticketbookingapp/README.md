# Lab 12 - ticketbookingapp (Conditional Rendering)

```
npx create-react-app ticketbookingapp
cd ticketbookingapp
# copy src/Guest.js, src/User.js, src/App.js
npm start
```
An element variable (`userComponent`) is set to either `<Guest />` or
`<User />` depending on the `isLoggedIn` state, toggled by the
Login/Logout buttons.
