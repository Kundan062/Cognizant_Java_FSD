# Lab 11 - eventexamplesapp (Event Handling)

```
npx create-react-app eventexamplesapp
cd eventexamplesapp
# copy src/CurrencyConvertor.js, src/App.js
npm start
```
Covers: multi-method click handler (Increment button increments the
counter AND logs a hello message), passing an argument to a handler
(Say Welcome), a synthetic click event (Press Me -> "I was clicked"),
and a `CurrencyConvertor` component that converts INR to EUR on
`handleSubmit`.
