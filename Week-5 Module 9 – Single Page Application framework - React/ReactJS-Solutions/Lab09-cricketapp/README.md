# Lab 09 - cricketapp (ES6 features)

```
npx create-react-app cricketapp
cd cricketapp
# copy src/ListofPlayers.js, src/IndianPlayers.js, src/App.js
npm start
```
Demonstrates `map()`, arrow-function `filter()`, array/variable
destructuring, and the spread operator to merge arrays. Toggle the
`flag` constant in `App.js` between `true`/`false` to switch which
component is displayed.
