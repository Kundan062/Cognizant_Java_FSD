# Lab 05 - CohortDetails Styling (CSS Modules)

```
npx create-react-app cohortapp
cd cohortapp
# copy src/CohortData.js, src/CohortDetails.js, src/CohortDetails.module.css, src/App.js
npm start
```
`box` class (300px wide, 10px margin, 10px top/bottom & 20px left/right
padding, 1px black border, 10px border-radius) is applied via the CSS
Module. `<dt>` elements use font-weight 500. The `<h3>` heading is green
when `status === "ongoing"` and blue otherwise.
